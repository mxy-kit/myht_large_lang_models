# myht_large_lang_models
This repository contains my experiments and implementations in NLP and LLMs.  Covered topics include:  Tokenizer training &amp; small LM (Byte-level BPE on Russian text)  Retrieval-Augmented Generation (RAG) with external knowledge sources  Parameter-Efficient Fine-Tuning (PEFT/LoRA) for sentiment classification  Alignment methods (DPO, PPO) for LLMs
